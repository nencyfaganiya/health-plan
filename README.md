# My Health Plan 🌿

A personal health companion app — daily routine tracker, meal planner, exercise plan, and progress history.

## Features
- Daily routine that changes weekday vs weekend
- Exercise plan with 3 phases that adapts as you progress
- Meal planner with shopping list
- Hourly water + walk reminders (weekdays)
- Push notifications for all tasks
- Progress history with calendar view
